Salih has written some code. He calls it a "proto-game", but in reality it is an unplayable version of Kalamala as it misses out one of the most important rules; that of groups. That said, we hope that it offers a tantalising glimpse into what is to come in term of a playable digital version of Kalamala.

NB. All of this work is done by Salih. Please blame Salih if you find any bugs (which I believe should now be gone), dodgy stylistic choices (which are a given with Salih), overcomplicated and inefficient and generally mind-bendingly-ugly code (which happens to be one of Salih's favourite hobbies), as well as for his unwillingness to impliment groups at the same time as the rest of the game.
